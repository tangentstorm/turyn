# Turyn Type Sequences (n ≤ 32)

Mirror of the Turyn Type sequence catalogue published by
**Hadi Kharaghani** (University of Lethbridge) at
<https://www.cs.uleth.ca/~hadi/research/TurynType/>.
All credit for the data goes to Kharaghani and collaborators;
this directory is just a local copy for offline use.

## Original notes (from `AddedNotes`)

> These are a complete list of Turyn Type sequences for n <= 32.
>
> Note that this list includes 66 sequences that were missed in our
> initial search that were later found by Stephen London.

## Layout

- `turyn-type-02` … `turyn-type-32` — complete lists of Turyn Type
  sequences at each even length `n`, including the 66 sequences
  later found by Stephen London.
- `turyn-32.txt` — text copy of the `n = 32` list.
- `OriginalFiles/sort02` … `sort32` — the original pre-London files
  (before the 66 missed sequences were added back).
